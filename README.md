# Axum Web Learning Project

This project is a personal training environment for learning modern web development in Rust. The goal is to build a clean, secure, and scalable backend using Axum and related tools while exploring real‑world concepts step by step.

## 🎯 Learning Goals

- Understand the fundamentals of **Axum** and its routing, handlers, layers, and error handling
- Learn **Axumate** for building structured and maintainable Rust web applications
- Work with **SQL databases** using async Rust (SQLx or SeaORM)
- Build RESTful APIs with proper request validation and response patterns
- Implement **authentication** (JWT, sessions, roles)
- Explore **web security** concepts such as input validation, rate limiting, and protection against common attacks
- Practice clean architecture, modular design, and testing in Rust
- Use environment configuration, Docker, and other development tools
